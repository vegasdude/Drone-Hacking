# Drone Hacking
### Defensive Security & Safety Research for UAV Systems

This repository focuses on **ethical, defensive, and educational security research** for unmanned aerial vehicle (UAV) and drone platforms.

The goal is to help developers, researchers, and operators better understand real-world attack surfaces and improve the resilience, safety, and reliability of drone systems.
